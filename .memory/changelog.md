# **היסטוריית שינויים ואירועים דחוסה (Changelog)**

* **הזרקת יחידה 1:** הושלמה הזרקת התוכן הרפואי מיחידה 1 ליצירת `unit1_physics.json` תחת דרישת ההפרדה המוחלטת (Decoupling) בין UI לתוכן.
* **הזרקת יחידה 2:** הושלמה הזרקת התוכן הרפואי מיחידה 2 ליצירת `unit2_pathologies.json`.
* **בקרת איכות התחלתית (QA):** עברה בהצלחה בדיקת עמידה במגבלות מילים (עד 40) ובטיחות רפואית (MS/Diabetes).
* **הקמת תשתית UI ל-Mobile-First (שלבים 1-2):** פיתוח מעטפת ה-`ReelContainer` ואנימציות Framer, בניית קומפוננטות הבלוקים וה-`ReelRenderer` שמשמש כ-Factory לנתוני ה-JSON, פתרון שגיאות Build והפעלת POC מלא בסביבת פיתוח מקומית.
* **הקמת תשתית Mobile-First והוכחת היתכנות:** פיתוח רכיבי המעטפת (ReelContainer, ReelRenderer), הוספת בלוקים גנריים, תיקוני Build של Next.js (use client) והפעלת שרת פיתוח המוכיח מערכת דינמית הניזונה מ-JSON.* **שלב הוכחת היתכנות של פיתוח ה-UI (MVP):** הוקמה מעטפת Mobile-First מלאה מבוססת Next.js ו-Framer Motion, שכללה בניית 5 קומפוננטות UI בסיסיות לרינדור JSON ופקטורי (`ReelRenderer`) והרצת שרת פיתוח להוכחת הפעלתן. תוקנו בעיות Build הקשורות לרכיבי צד-לקוח.
