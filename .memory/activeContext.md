# **הקשר רגעי ופעיל (Sliding Window Temporal Memory)**

## **מוקד הפעילות הנוכחי (Current Focus)**

המשימה הנוכחית: פיתוח אנימציות חיות מבוססות חוקי הפיזיקה (Phase 3). המערכת עלתה לפרודקשן (Next.js + Tailwind + Framer Motion) עם 5 בלוקי למידה דינמיים הניזונים מ-JSON. שדרוג ה-`AnimationFactory` נמשך, כעת ממוקד באנימציות אינטראקטיביות וחישוב כוחות דינמיים (CG/CB). ארכיטקטורת המידע שודרגה לתמיכה ברצפי למידה (Sequences) עם כותרות קפואות.

## **חלון עשרת האירועים האחרונים (Sliding Window)**

1. בקרת איכות (QA) עברה בהצלחה למגבלות מילים (40) ובטיחות רפואית (MS/Diabetes).
2. הקמת מעטפת ה-Mobile-First (`ReelContainer`) ואנימציות Framer.
3. בניית 5 קומפוננטות ה-Blocks הגנריות תחת `components/blocks`.
4. פיתוח ה-`ReelRenderer` שמשמש כפקטורי להצגת הנתונים מה-JSON ל-Blocks.
5. תיקוני Build ב-Next.js (הזרקת `use client` לרכיבי UI).
6. הפעלת שרת פיתוח (`npm run dev`) עם הוכחת היתכנות מלאה למערכת הדינמית.
7. יצירת Implementation Plan לשלב 3: פיתוח אנימציות פיזיקה ליבתיות והגדרת Fallback.
8. עדכון ה-Memory Bank למעקב אחרי 18 ה-Visual Triggers הנותרים שמהווים חוב טכני לעתיד.
9. שדרוג אנימציית MetacentricTorque למערכת אינטראקטיבית המדמה קטיעות גפיים והשפעתן על מרכז הכובד והציפה.
10. שדרוג הארכיטקטורה וה-UI: תמיכה ברצפי רילסים (Sticky Sequence Titles) והבלטת תגית "דוגמה קלינית".
