# **הקשר רגעי ופעיל (Sliding Window Temporal Memory)**

## **מוקד הפעילות הנוכחי (Current Focus)**

המשימה הנוכחית: פיתוח אנימציות חיות מבוססות חוקי הפיזיקה (Phase 3). המערכת עלתה לפרודקשן (Next.js + Tailwind + Framer Motion) עם 5 בלוקי למידה דינמיים הניזונים מ-JSON. כעת המוקד הוא שדרוג ה-`AnimationFactory` ומעקב אחרי 18 אנימציות חסרות ב-Backlog.

## **חלון עשרת האירועים האחרונים (Sliding Window)**

1. הזרקת התוכן הרפואי מיחידה 1 ליצירת `unit1_physics.json` (הפרדה מוחלטת).
2. הזרקת התוכן הרפואי מיחידה 2 ליצירת `unit2_pathologies.json`.
3. בקרת איכות (QA) עברה בהצלחה למגבלות מילים (40) ובטיחות רפואית (MS/Diabetes).
4. הקמת מעטפת ה-Mobile-First (`ReelContainer`) ואנימציות Framer.
5. בניית 5 קומפוננטות ה-Blocks הגנריות תחת `components/blocks`.
6. פיתוח ה-`ReelRenderer` שמשמש כפקטורי להצגת הנתונים מה-JSON ל-Blocks.
7. תיקוני Build ב-Next.js (הזרקת `use client` לרכיבי UI).
8. הפעלת שרת פיתוח (`npm run dev`) עם הוכחת היתכנות מלאה למערכת הדינמית.
9. יצירת Implementation Plan לשלב 3: פיתוח אנימציות פיזיקה ליבתיות והגדרת Fallback.
10. עדכון ה-Memory Bank למעקב אחרי 18 ה-Visual Triggers הנותרים שמהווים חוב טכני לעתיד.
