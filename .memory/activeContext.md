# **הקשר רגעי ופעיל (Sliding Window Temporal Memory)**

## **מוקד הפעילות הנוכחי (Current Focus)**

המשימה הנוכחית: פיתוח אנימציות חיות מבוססות חוקי הפיזיקה (Phase 3). המערכת עלתה לפרודקשן (Next.js + Tailwind + Framer Motion) עם 5 בלוקי למידה דינמיים הניזונים מ-JSON. שדרוג ה-`AnimationFactory` נמשך, כעת ממוקד באנימציות אינטראקטיביות וחישוב כוחות דינמיים (CG/CB).

## **חלון עשרת האירועים האחרונים (Sliding Window)**

1. הזרקת התוכן הרפואי מיחידה 2 ליצירת `unit2_pathologies.json`.
2. בקרת איכות (QA) עברה בהצלחה למגבלות מילים (40) ובטיחות רפואית (MS/Diabetes).
3. הקמת מעטפת ה-Mobile-First (`ReelContainer`) ואנימציות Framer.
4. בניית 5 קומפוננטות ה-Blocks הגנריות תחת `components/blocks`.
5. פיתוח ה-`ReelRenderer` שמשמש כפקטורי להצגת הנתונים מה-JSON ל-Blocks.
6. תיקוני Build ב-Next.js (הזרקת `use client` לרכיבי UI).
7. הפעלת שרת פיתוח (`npm run dev`) עם הוכחת היתכנות מלאה למערכת הדינמית.
8. יצירת Implementation Plan לשלב 3: פיתוח אנימציות פיזיקה ליבתיות והגדרת Fallback.
9. עדכון ה-Memory Bank למעקב אחרי 18 ה-Visual Triggers הנותרים שמהווים חוב טכני לעתיד.
10. שדרוג אנימציית MetacentricTorque למערכת אינטראקטיבית המדמה קטיעות גפיים והשפעתן על מרכז הכובד והציפה.
